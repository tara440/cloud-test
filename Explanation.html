Explain Storage virtualization

Virtualization refers to the presentation of a simple file, logical volume or other storage object
to an application in such a way that allows the physical complexity of the storage to be hidden from
the storage administrator and the application. Today there are many kinds of virtualization such as Server
virtualization, Network Virtualization, Client and desktop virtualization, Services and Applications
Virtualization and Storage virtualization, in this project we are focus on storage virtualization.

Storage Virtualization

This type of virtualization creates the abstraction of logical storage from physical storage,
so the user can access the integrated storage resources through unified logical interface of
this view. In a way using this technique, users do not have to be worried about the specific
location of their data, which can be identified using a logical path. Storage virtualization
grants us to harness a broad scope of storage facilities and represent them under a single
logical file system. There are various techniques for storage virtualization, one of the most
suitable being network-based virtualization by means of Storage Area Networks (SANs). SANs
use a network-accessible device in away large bandwidth connection to provide storage facilities.
Generally, there are two type of Storage Virtualization

Block Level Storage Virtualization

This type of storage virtualization is implemented in Storage Area Network, and it provides
a translation layer in the SAN, between the hosts and the storage arrays.In the storage device
the virtualization entity is map those virtual blocks to physical block, the storage capacity

File level storage virtualization

It is implemented Network Attach Storage level and help in solving NAS problems by removing
the dependencies between the data accessed at the file level and the location where the files
are physically stored, in this type of storage virtualization the virtual storage is provides
by virtualization entity is accessible to Operating System or applications in the form of file
and directories the example of it like NTFS file system in windows 2000 & NT.
